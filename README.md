# Portfolio Project

A personal portfolio website made with HTML, CSS and JavaScript.

## Description

This project is a static portfolio website that showcases different projects and activities. The site has multiple pages for each section.

## Project Files

- `index.html` - Homepage
- `chant.html` - Singing page
- `cours.html` - Teaching page
- `ecris.html` - Writing page
- `photos.html` - Photography page
- `styles.css` - CSS file with all the styles
- `script.js` - JavaScript file for the burger menu
- `images/` - Folder with the website images

## How to use

1. Open the `index.html` file in your browser
2. Navigate between different pages using the menu
